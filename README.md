# File Analyser — Статистичний аналіз розмірів файлів

Аналіз розподілу розмірів файлів файлової системи з побудовою графіків і PDF-звіту.

## Що демонструє цей проєкт

- Завантаження та статистичний аналіз великих числових масивів (542 798 файлів)
- Побудова логарифмічної гістограми, ECDF, зонального та кругового графіків
- Генерація PDF-звіту з таблицями, графіками та аналізом через `reportlab`
- Знаходження найвужчого діапазону, що охоплює 90% файлів

## Структура

```
file_analyser/
├── analyze_files.py      # Статистичний аналіз і побудова графіків
├── generate_report.py    # Генерація PDF-звіту
├── file_sizes_raw.txt    # Вхідні дані: розміри файлів у байтах (по одному на рядок)
├── report_plots/         # Згенеровані графіки (PNG)
│   ├── fig1_log_histogram.png
│   ├── fig2_zones.png
│   ├── fig3_ecdf.png
│   └── fig4_pie.png
└── file_size_report.pdf  # Готовий PDF-звіт
```

## Як запустити

### 1. Статистичний аналіз і побудова графіків

```bash
python3 analyze_files.py
```

Виведе у консоль статистику та збереже графіки у `report_plots/`:


### 2. Генерація PDF-звіту

```bash
python3 generate_report.py
```

Створить `file_size_report.pdf` з усіма графіками та таблицями.

### Підготовка власних вхідних даних

Щоб проаналізувати свою файлову систему, зібери розміри файлів командою:

```bash
find / -type f -printf "%s\n" 2>/dev/null > file_sizes_raw.txt
```

або для конкретної директорії:

```bash
find /home -type f -printf "%s\n" 2>/dev/null > file_sizes_raw.txt
```
